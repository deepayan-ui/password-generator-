# 🔐 KeyGen - Simple Password Generator

A clean, modern, and lightweight password generator built using **HTML, CSS, and JavaScript**. It allows users to instantly generate secure passwords with customizable options.

## 📸 Preview

Generate strong passwords with:
- Adjustable password length
- Include numbers
- Include symbols
- One-click copy to clipboard

---

## ✨ Features

- 🔑 Generate random secure passwords
- 📏 Choose password length (4–32 characters)
- 🔢 Option to include numbers
- 🔣 Option to include special symbols
- 📋 Copy password to clipboard with one click
- 🎨 Modern responsive UI
- ⚡ Fast and lightweight
- 💻 No external libraries required

---

## 🛠️ Technologies Used

- HTML5

## 🔒 Password Generation Logic

The application generates passwords by:

- Creating a pool of characters
- Adding letters by default
- Optionally adding:
  - Numbers
  - Symbols
- Randomly selecting characters from the pool until the desired length is reached.
